# daily-log

A public learning log auto-generated every day via GitHub Actions.

Each entry rotates through four categories:
- **NLP terms** — vocabulary from my MS AI coursework
- **LC patterns** — LeetCode problem-solving patterns
- **CS concepts** — distributed systems, architecture, fundamentals
- **Reflections** — open-ended prompts to think through

Entries live in `logs/YYYY-MM/YYYY-MM-DD.md`.
